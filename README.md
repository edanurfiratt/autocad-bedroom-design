# AutoCAD Bedroom Design

A bedroom floor plan designed using AutoCAD as part of an engineering design assignment.

## Project Overview

This project includes the 2D design of a bedroom with furniture placement, dimensions, room labeling, and hatch patterns. The drawing was created using AutoCAD and follows basic technical drawing principles.

## Room Information

- Room Type: Bedroom
- Area: 16.5 m²

## Features

- Wall layout design
- Door opening representation
- Furniture placement
  - Bed
  - Wardrobe
  - Storage units
- Room labeling
- Dimensioning
- Hatch patterns
- Technical floor plan presentation
